# Last Colony: Decisions — Public Store Pages

This repo contains the public pages required by Google Play and App Store Connect for `Last Colony: Decisions`.

## Pages

- Home: GitHub Pages site root
- Privacy Policy: `/privacy/`
- Support URL: `/support/`

After creating the GitHub repo, enable `Settings → Pages → Source: GitHub Actions`.

## Store assets

- Google Play feature graphic: `assets/google-play-feature-graphic.png`
- Web-render smoke screenshots: `screenshots/`

The app itself is not included here. This repo is only for the public support/privacy website and store listing assets.
